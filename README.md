# 🍋 Sunnyside Agency Landing Page

ეს არის კრეატიული სააგენტოს სადესანტო გვერდი (Landing Page), რომელიც გამოირჩევა ნათელი ფერებით, თანამედროვე გრაფიკითა და დახვეწილი სტრუქტურით. პროექტი ფოკუსირებულია ვიზუალურ ნარატივსა და სრულ ადაპტაციაზე (Responsive Design).

🌐 **Live Demo:** [endearing-swan-36d433.netlify.app](https://endearing-swan-36d433.netlify.app/)

## ✨ პროექტის მახასიათებლები

- **Custom Navigation:** ინტერაქტიული მენიუ, რომელიც მორგებულია მობილურ ვერსიაზეც.
- **Hero Section:** ეფექტური მთავარი სექცია მოძრავი (Arrow) ელემენტებით.
- **Grid Layouts:** რთული ვიზუალური ბადე, სადაც ტექსტი და გამოსახულებები იდეალურადაა ბალანსირებული.
- **Client Testimonials:** მომხმარებელთა გამოხმაურებების სექცია სუფთა და პროფესიონალური დიზაინით.
- **Photo Gallery:** სრულად ადაპტირებული ფოტო გალერეა ფუტერის ზემოთ.

## 🛠️ გამოყენებული ტექნოლოგიები

- **HTML5:** სემანტიკური მარკაპისთვის.
- **CSS3 (Flexbox & Grid):** რთული დიზაინისა და პოზიციონირებისთვის.
- **JavaScript:** მობილური მენიუს (Hamburger menu) ფუნქციონირებისთვის.
- **SVG Graphics:** მაღალი ხარისხის ლოგოებისა და იკონებისთვის.
- **Netlify:** პროექტის სწრაფი და უსაფრთხო ჰოსტინგისთვის.

## 📁 სტრუქტურა

- `index.html` - ვებ-გვერდის ძირითადი კოდი.
- `images/` - ყველა გრაფიკული აქტივი (Logo, Icons, Photos).
- `style.css` - დიზაინის და ანიმაციების ფაილი.
